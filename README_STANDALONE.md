# 予算管理アプリ (スタンドアロン・ポータブル版)

このバージョンは、Node.jsなどのインストールが一切不要な、単一の実行ファイル形式です。

## 🚀 使い方

1. `Budget Performance Manager x.x.x.exe` をダブルクリックします。
2. アプリケーションが直接起動します。
3. 初回起動時に、Windowsから「保護されました」と表示される場合は、「詳細情報」→「実行」をクリックしてください。

## 📦 特徴

- **インストール不要**: 実行ファイル一つで動きます。
- **Node.js不要**: 必要な環境はすべて内蔵されています。
- **オフライン動作**: 依存関係のダウンロードなどは不要です。

## ⚠️ 注意事項

- ファイルサイズが大きめ（約150MB〜200MB）ですが、これはアプリケーションを動かすための全ての部品が含まれているためです。
- データの保存先などはアプリ内の指示に従ってください。
