# Budget Performance Manager - インストール不要版

## 使い方

### 初回セットアップ
1. このフォルダ全体を任意の場所にコピー
2. `install.bat` をダブルクリック（初回のみ）
3. インストールが完了するまで待つ（数分かかります）

### アプリの起動
`start.bat` をダブルクリック

ブラウザが自動的に開き、アプリが起動します。

### アプリの終了
起動時に開いたコマンドプロンプトウィンドウを閉じてください。

## 必要な環境
- Windows 10/11
- インターネット接続（初回インストール時のみ）

## トラブルシューティング

### Node.jsがインストールされていない場合
1. https://nodejs.org/ にアクセス
2. LTS版をダウンロードしてインストール
3. PCを再起動
4. `install.bat` を再実行

### ポートが使用中のエラーが出る場合
他のアプリがポート3000を使用している可能性があります。
そのアプリを終了してから再度 `start.bat` を実行してください。

### ブラウザが自動で開かない場合
手動で http://localhost:3000 にアクセスしてください。
