# 部門別財務分析機能 - 実装完了

## 概要
各部門別の詳細な財務分析機能を実装しました。18種類の財務指標を計算し、ダッシュボードとRatioテーブルで可視化します。

## 実装された機能

### 1. 部門別ダッシュボード (`Dashboard 各部` タブ)

#### 主な機能:
- **従業員数管理**: 各部門の従業員数を設定・編集可能
- **KPIカード**: 4つの主要指標を一目で確認
  - 営業利益率
  - 売上成長率
  - 一人当たり売上高
  - 損益分岐点比率
- **月次推移グラフ**: 粗利率、営業利益率、販管費率の月次トレンド
- **総合評価レーダーチャート**: 5つの指標を視覚化
- **部門間ベンチマーク比較**: 全部門の主要指標を横並びで比較
- **AI分析アラート**: 自動生成される改善提案と評価

### 2. 部門別Ratioテーブル (`各部Ratio` タブ)

#### 表示される18の財務指標:

##### 収益性指標 (Profitability)
1. **売上総利益率（粗利率）**: 売上に対する粗利益の割合
2. **営業利益率**: 売上に対する営業利益の割合
3. **経常利益率**: 売上に対する経常利益の割合

##### 費用効率 (Cost Efficiency)
4. **販売費及び一般管理費率**: 売上に対する販管費の割合
5. **人件費率**: 売上に対する人件費の割合
6. **広告宣伝費率**: 売上に対する広告宣伝費の割合

##### 成長性 (Growth)
7. **売上成長率**: 前年比の売上成長率
8. **営業利益成長率**: 前年比の営業利益成長率

##### 営業効率 (Productivity)
9. **一人当たり売上高**: 従業員一人当たりの売上高
10. **一人当たり営業利益**: 従業員一人当たりの営業利益

##### 利益構造の健全性 (Quality of Earnings)
11. **変動費率**: 売上に対する変動費の割合
12. **固定費率**: 売上に対する固定費の割合
13. **損益分岐点比率**: 損益分岐点売上高÷実際売上高

##### 高度分析 (Advanced)
14. **営業レバレッジ**: 売上総利益÷営業利益
15. **利益感応度**: 営業レバレッジ×売上変動率

#### 表示形式:
- **月次データ**: 4月〜3月の各月の値
- **期間データ**: 上期(1H)、下期(2H)、年間(FY)
- **カラーコーディング**: 値の良し悪しを色で表示
- **固定列**: 横スクロール時も指標名は常に表示
- **詳細説明**: 各指標の計算式と意味を解説

## 使用方法

### 1. 部門の選択
左サイドバーの「Department Filter」から分析したい部門を選択します。

### 2. 従業員数の設定
1. 「Dashboard 各部」タブを開く
2. 右上の従業員数欄の「編集」ボタンをクリック
3. 人数を入力して「保存」

### 3. ダッシュボードの確認
- KPIカードで主要指標を確認
- AIアラートで改善点をチェック
- グラフで月次トレンドを分析
- ベンチマークで他部門と比較

### 4. 詳細比率の確認
「各部Ratio」タブで18の財務指標を詳細に分析できます。

## AI分析アラートの種類

システムは以下の条件で自動的にアラートを生成します:

### ✅ 優良評価
- 営業利益率が15%以上
- 売上が前年比10%以上成長
- 損益分岐点比率が70%未満

### ⚠️ 注意
- 営業利益率が5%未満
- 販管費率が30%以上
- 一人当たり売上高が1,000万円未満
- 営業レバレッジが5倍以上

### 🔴 危険
- 売上が前年比5%以上減少
- 損益分岐点比率が90%以上

### 🚀 好調
- 売上が前年比10%以上成長

## 計算式の詳細

### 損益分岐点比率
```
損益分岐点売上高 = 固定費 ÷ 粗利率
損益分岐点比率 = 損益分岐点売上高 ÷ 実際売上高 × 100
```

### 営業レバレッジ
```
営業レバレッジ = 売上総利益 ÷ 営業利益
```

### 利益感応度
```
利益感応度 = 営業レバレッジ × 売上変動率
```

## ベンチマーク基準

### 営業利益率
- 優良: 10%以上
- 標準: 5-10%
- 要改善: 5%未満

### 販管費率
- 優良: 20%以下
- 標準: 20-30%
- 要改善: 30%以上

### 損益分岐点比率
- 安全: 70%以下
- 標準: 70-85
- 危険: 90%以上

### 一人当たり売上高
- 優良: 3,000万円以上
- 標準: 800万円以上
- 要改善: 600万円未満

## 注意事項

1. **従業員数の設定**: 生産性指標（一人当たり売上高・利益）を正確に計算するため、必ず従業員数を設定してください。
2. **前年データ**: 成長率を計算するため、前年データがアップロードされている必要があります。
3. **コード体系**: 財務データのコード体系が標準的な勘定科目コードと一致していることを前提としています。

## サポートされるコード

- 9821: 売上高
- 9861: 営業利益
- 9865: 経常利益
- 9701: 人件費
- 9711: 広告宣伝費
- COGS: 売上原価（キーワード検索）
- SGA: 販売費及び一般管理費（キーワード検索）
